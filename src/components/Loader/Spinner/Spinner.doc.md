#### Spinner component

```jsx
<div>
    <Spinner size="sm" />
    <Spinner size="md" />
    <Spinner size="lg" />
</div>
```

#### Spinner with another color

```jsx
<div>
    <Spinner size="sm" color="orange" />
    <Spinner size="md" color="orange" />
    <Spinner size="lg" color="orange" />
</div>
```
